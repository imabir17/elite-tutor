EliteTutor Documentation

Thank you for purchasing the EliteTutor Next.js Academic Template! This highly polished, animated landing page is designed for high-end tutors and educational consultants.

This template is built using the latest bleeding-edge React features including Next.js 14+ (App Router), Tailwind CSS v4, and Framer Motion.

Prerequisites

Before you begin, ensure you have the following installed on your local development environment:

  • Node.js - Version 18.17.0 or higher.
  • npm (or yarn, pnpm) - Usually comes with Node.js.
  • A text editor of your choice (VS Code is highly recommended).

Installation & Setup

Getting your project running locally is extremely simple. Open your terminal, navigate to the template folder, and run the following commands:

  1. Install all dependencies: 
    npm install
  2. Start the local development server: 
    npm run dev

Note: Once the server is running, open your browser and navigate to http://localhost:3000. Any changes you make to the source code will automatically refresh the page!

Project Structure

This template follows the official Next.js App Router conventions. Here is a breakdown of where to find everything:

.
├── public/                 # Static assets (images, favicons)
├── src/
│   ├── app/                # Next.js App Router pages and layouts
│   │   ├── globals.css     # Global Tailwind v4 styles and CSS variables
│   │   ├── layout.tsx      # Root HTML layout, Fonts mapping, and Providers
│   │   └── page.tsx        # Main Landing Page assembling all components
│   ├── components/         # React Components
│   │   ├── layout/         # Header and Footer
│   │   ├── sections/       # Hero, Services, Pricing, Stats, Testimonials
│   │   └── ui/             # Reusable UI elements (Modal, Buttons)
│   ├── context/            # React Contexts (e.g., Modal Context for forms)
│   └── lib/                
│       ├── data.ts         # CENTRALIZED DATA: Edit this file to change site text!
│       └── utils.ts        # Helper functions (clsx, tailwind-merge)
└── package.json            # Project dependencies

Customizing Content

To make editing fast, all the dynamic data (pricing details, statistics, testimonials, and service frameworks) is centralized in a single file.

Navigate to src/lib/data.ts.

In this file, you will find simple JavaScript arrays and objects. For example, to edit the pricing tiers, locate the pricingTiers array:

export const pricingTiers = [
  {
    id: "starter",
    name: "Crash Course",
    monthlyPrice: 199,
    fullPrice: 499,
    features: [
      "10 Hours 1-on-1 Tutoring",
      "2 Full Mock Tests"
    ],
    isPopular: false,
  },
  // ...
]

Images: Placeholder images (like the portrait in the Hero section) are hardcoded directly into the component. To change the Hero image, open src/components/sections/Hero.tsx and replace the <img src="..." /> URL with your own image path located in the public folder (e.g., src="/my-photo.jpg").

Styling & Theme (Tailwind v4)

Because this template uses Tailwind CSS v4, there is no traditional tailwind.config.ts file. The core theme colors, custom fonts, and custom utilities are defined directly using CSS variables in src/app/globals.css.

To change the primary color scheme, edit these variables:

@theme inline {
  --color-primary: #0F172A; /* Main dark blue */
  --color-accent: #F59E0B;  /* Golden Call-To-Action color */
  --color-background: #FAFAFA; /* Off-white page background */
  --color-foreground: #0F172A; /* Default text color */
}

All Google Fonts are imported optimally in Next.js via src/app/layout.tsx. If you wish to change the fonts, alter the imports in the layout file.

Animations (Framer Motion)

This template heavily utilizes framer-motion. The animations are primarily controlled via two patterns:

  1. Scroll-Driven Appearance: Components fade up natively using whileInView and viewport={{ once: true }} attributes on <motion.div> tags.
  2. Parallax & Scroll Progress: The Methodology timeline and Hero parallax rely on the useScroll() and useTransform() hooks. Modifying these hooks inside the components allows you to adjust the speed or distance of the parallax effects.

Deployment

Your Next.js project is fully optimized for production deployment on exactly the infrastructure of your choice, though Vercel is recommended.

  • Create a free account on Vercel.com.
  • Push your locally modified code to a GitHub repository.
  • Log into Vercel, click "Add New Project", select your repository, and click Deploy.

If deploying to a traditional host, build the application manually via:

npm run build
npm run start
